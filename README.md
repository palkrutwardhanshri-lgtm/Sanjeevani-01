## 🚀 The Mission 
At **Sanjeevani Care**, we believe that pharmaceutical delivery should be as seamless, transparent, and empathetic as healthcare itself. Modern pharmacies and clinics struggle with opaque supply chains, delayed deliveries, and fragmented patient communication. 

**Our Solution?** A blazing-fast, immersive, and fully responsive platform that bridges the gap between healthcare providers and patients—powered by cutting-edge web technologies like React, GSAP, and Three.js.

---

## 🧩 Deep Dive Into Our Platform

Sanjeevani isn't just a basic dashboard; it's a complete ecosystem designed for scale, speed, and reliability.

### 1️⃣ Advanced Logistics Dashboard
- **Real-Time Tracking:** Visualize delivery fleets and monitor live order delivery statuses.
- **Data-Driven Insights:** Get actionable intelligence on delivery times, conversion rates, and performance bottlenecks.
- **Interactive UI:** Smooth, data-rich visualizations built to handle enterprise-level healthcare operations.

### 2️⃣ Omnichannel Patient Communication
- **WhatsApp Integration:** Keep patients updated on their prescriptions right where they already chat.
- **SMS & Voice Modules:** Automated alerts and calls to ensure critical medical deliveries are never missed by the patient.
- **Empathetic Engagement:** Notifications styled and timed to reduce patient anxiety regarding their crucial medications.

### 3️⃣ Award-Winning Immersive User Experience
- **Cinematic Web Design:** Utilizes `GSAP` and `Lenis` for silky-smooth scroll choreography and mask-reveal animations across all pages.
- **3D Interactive Elements:** Built with `Three.js` (`@react-three/fiber`) and `Vanta.js` to create a modern, trust-building brand experience that wows stakeholders.
- **Mobile-First Approach:** Our custom device layouts prove that our app looks spectacular on desktop, tablet, and mobile.

## ⚙️ Get Up and Running Locally

Want to experience Sanjeevani on your machine? You can have the platform running in under two minutes!

### Prerequisites
Make sure you have **[Node.js](https://nodejs.org/)** installed on your machine.

### Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/Sanjeevani.git
   cd Sanjeevani
   ```

2. **Install dependencies:**
   ```bash
   # We use npm to manage our powerful animation and UI libraries
   npm install
   ```

3. **Ignite the development server:**
   ```bash
   npm run dev
   ```
   *Your terminal will give you a local link (e.g., `http://localhost:5173/`). Click it to view the magic!*
